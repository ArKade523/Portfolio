I built this arcade cabinet at the end of 8th grade. It is run on a Raspberry Pi 3B, with emulators loaded on an open-source, Linux-based operating system. This project was my first time combining woodworking, rudimentary software engineering, and rudimentary computer engineering. To this day, it is still one of my favorite projects to ever have worked on.
This is a Hydrogen Powered Vehicle. I designed and built it in 10th grade to be as efficient as possible.
This is the team I built my Hydrogen Powered Vehicle with. We also collaborated on making a Solar Powered Vehicle, using the same chassis. This project was also useful for learning the basics of circuitry.
